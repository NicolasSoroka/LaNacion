
# La Nacion challenge


## Instalación ⚙

Clonar repo

```bash
  git clone https://github.com/NicolasSoroka/LaNacion.git
  cd la-nacion
```
Instalar dependencias y correr el proyecto
  ```bash
  npm i
  npm run dev
```
